Pure

A modern and intuitive database manager

About the Project

Pure is a database manager developed with Wails and React-TypeScript. This desktop application allows you to connect to your PostgreSQL databases, explore their structures, and execute SQL queries with a modern and intuitive user interface.

Technologies Used

  • Wails - Framework for creating desktop applications with Go and web interfaces
  • React & TypeScript - For a reactive and typed user interface
  • Shadcn UI - Library of modern and accessible UI components
  • Go - For a robust and high-performance backend
  • PostgreSQL - Support for PostgreSQL databases

Features

Connection Management

Add, test, and organize your database connections

Secure Storage

Your connection information is stored locally in a secure manner

Database Exploration

Easily browse your databases and their tables

SQL Editor

Integrated SQL editor with auto-completion to execute your queries

Installation

You can download compiled versions for your operating system from the Releases section of our GitHub repository.

For developers wishing to contribute or compile the application:

  1. Clone the repository: git clone https://github.com/USERNAME/Pure.git
  2. Install frontend dependencies: cd frontend && npm install
  3. Launch the application in development mode: wails dev
